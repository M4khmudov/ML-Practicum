Сравнительный анализ моделей детекции дорожных знаков

Данный репозиторий содержит комплексное CV-решение, разработанное в рамках учебной практики по направлению "Высокопроизводительные вычислительные системы". Основная задача проекта — провести обучение и последующее сопоставление пяти известных архитектур для решения проблемы локализации и классификации дорожных объектов на изображениях.

| Модель           | Ключевая особенность                                      | Backbone        | Исходные веса             |
| ---------------- | --------------------------------------------------------- | --------------- | ------------------------- |
| **YOLOv8**       | предсказание без привязок (anchor-free), единый проход    | CSPDarknet      | `yolov8s.pt`              |
| **YOLOv5u**      | безанкерная версия YOLOv5 (аналогична YOLOv8 по голове)   | CSPDarknet      | `yolov5su.pt`             |
| **Faster R-CNN** | двухэтапная схема: сначала регионы, затем классификация   | ResNet-50 + FPN | `fasterrcnn_resnet50_fpn` |
| **SSD300**       | многомасштабные карты признаков, привязки (anchor-based)  | VGG-16          | `ssd300_vgg16`            |
| **RetinaNet**    | фокальная функция потерь для борьбы с дисбалансом классов | ResNet-50 + FPN | `retinanet_resnet50_fpn`  |


    Важное уточнение: В проекте задействована версия YOLOv5u, а не оригинальный YOLOv5. Буква "u" в названии указывает на использование безанкерной головы, аналогичной той, что применяется в YOLOv8, тогда как классическая реализация опирается на предопределённые анкеры.

Настройка окружения

Пайплайн рассчитан на использование GPU-ускорителей. Рекомендуется предварительно установить драйверы CUDA.
bash

# 1. Установка PyTorch с поддержкой CUDA (пример для GTX 1660 Super, CUDA 12.x)
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121

# 2. Установка остальных зависимостей
pip install -r requirements.txt

# 3. Проверка доступности GPU
python -c "import torch; print('CUDA:', torch.cuda.is_available())"

Порядок работы
bash

# Конвертация исходного датасета GTSDB из data/raw/ в требуемые форматы (YOLO + JSON-манифест)
python main.py --mode prepare --config configs/default.yaml

# Запуск обучения для конкретной модели
python main.py --mode train --model yolov8 --config configs/default.yaml

# Запуск обучения для всех пяти моделей последовательно
python main.py --mode train --model all --config configs/default.yaml

# Сравнительная оценка обученных моделей (формирование таблиц и графиков в директории results/)
python main.py --mode eval --model all --config configs/default.yaml

# Выполнение инференса на произвольном изображении
python main.py --mode predict --model yolov8 --source path/to/image.jpg

Используемые данные

GTSDB (German Traffic Sign Detection Benchmark) — набор данных, содержащий 900 кадров с размеченными дорожными знаками, охватывающих 43 класса объектов.

Файлы изображений и сопутствующая разметка (gt.txt) уже размещены в директории data/raw/ и готовы к обработке.
